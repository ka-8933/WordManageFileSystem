一款基于 Java + HTML/CSS/JavaScript 开发的全栈单词管理系统，专注于简洁易用的单词学习、管理与搜索体验，支持用户信息管理、头像上传、报表生成等核心功能。

## 📋 项目介绍
本项目旨在打造轻量、高效的单词管理工具，解决传统单词软件操作复杂、交互不友好的问题。核心优化点包括：
- ✅ 移除冗余分页逻辑，提升单词搜索响应速度
- ✅ 修复输入框显示 `undefined` 异常问题
- ✅ 简洁的用户界面，兼顾移动端/桌面端适配
- ✅ 完整的用户信息管理、单词增删改查功能
